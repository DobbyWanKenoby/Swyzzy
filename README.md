#  Swyzzy

Swyzzy - социальная сеть, основанная на ведении списка желаний пользователей.

## Используемый стек технологий

- Swift
- MVP + `SwiftCoordinatorsKit`
- `Swinject`
- `UIKit` + `SnapKit`
- `Combine`
- `Firebase` (`Auth`, `Firestore`)
- iOS 15+
